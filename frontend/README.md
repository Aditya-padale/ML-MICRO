# Frontend (React + Material UI)

## Setup

```bash
# Using npm
cd frontend
npm install
# or using yarn
# yarn
```

Set API base URL (optional):
- CRA: set REACT_APP_API_BASE
- Vite: set VITE_API_BASE

## Run
```bash
npm start
```

This expects the backend running at http://localhost:8000 by default.
